Projects

Space Shooter

After wanting to deepen my understanding of game development, I built a 2D space shooter in C++ using SFML. The game simulates player movement, enemy AI, shooting mechanics, and collision detection in a modular, object-oriented architecture. I engineered features like dynamic background titling, directional shooting based on mouse position, health and scoring systems, and enemy destruction animations.

Reddit Bot

Built a Reddit bot using Python and the PRAW API. I used asynchronous scripting and handled rate-limiting logic manually. The bot monitored subreddits and responded based on sentiment analysis.

Route

Crafted a word-ladder puzzle where players transform a start word into a target word by changing one letter at a time. Built a Node.js script to generate a trie-based dictionary for fast, real-time word validation. On the front end, implemented responsive UI with animated feedback for each guess, hint and “how to play” modal dialogs, and a guess tracking system showing both current progress and target metrics. Deployed on my website with live demo and open-source repository.
